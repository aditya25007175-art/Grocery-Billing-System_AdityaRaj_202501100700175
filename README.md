## Problem Statement
A grocery store wants to calculate the total cost of items purchased by a customer.
- Calculate total cost of 3 items.
- Add 10% discount if total exceeds $50.
- Display final payable amount.

---

## Approach
1. Take input for 3 item prices.
2. Calculate total.
3. Apply 10% discount if total > 50.
4. Display original total, discount, and final amount.
